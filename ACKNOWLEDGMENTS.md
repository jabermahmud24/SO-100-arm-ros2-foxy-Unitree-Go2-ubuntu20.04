## so_arm_100_hardware

This repository includes work derived from:

- Bruk G., `so_arm_100_hardware`: https://github.com/brukg/so_arm_100_hardware

The original package provides the ROS 2 Control hardware interface for the SO-ARM100 robotic manipulator. It supports direct serial communication with the physical robot and ROS topic-based communication for simulation.

Modifications/adaptations in this repository are intended to support ROS 2 Foxy, Ubuntu 20.04, and Unitree Go2 integration.
